# Setting up & Running

1. `npm install`
2. `npm run start`

# Debugging

To automatically re-run when file changes are made use:

`DEBUG=jira-sprint-summary:* npm run devstart`

# Using Tool

Access the tool at `http://localhost:3000/stats`
